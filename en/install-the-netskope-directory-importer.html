Install the Netskope Directory Importer

  1. Generate a REST API token from the Netskope tenant. Go to Settings > Tools > REST API and click Generate New Token.

  2. Click Generate.

  3. Refresh the page to view the generated token.

  4. You have the option to set Token Expiry. Keep the default value Never.

    Important

    If token expiry is set for an amount of time, then after expiry a new token needs to be generated. The Token is used in AD Importer and REST API calls to download events, which will stop working on token expiry until new token details has been generated and updated in AD Importer and REST API configurations.

  5. Download the Directory Importer tool from the Netskope tenant. Go to Settings > Tools > Directory Tools and on the On-Prem Integration tab click Download Tools.

    Also copy the Addon URL, which is required to configure the Directory Importer.

  6. Login to the Windows device with local admin credentials.

  7. Install Microsoft Visual C++ 2010 SP1 Redistributable package (x64).

  8. Launch the NSAdapters.msi file to install Netskope Directory Importer. When the setup wizard opens, click Next.

  9. Leave the username and password blank and click Next.

  10. Select Directory Importer and click Install.

  11. Click Finish

  12. Launch Services.msc.

  13. Right-click Netskope Directory Importer Service and select Properties.

  14. Select the Log On Tab and clickBrowse. Search for the local user administrator account.

  15. Enter the password and click Apply.

  16. This will register the local admin account with the Log On as a Service privilege. Click OK and close the services console.

  17. The Netskope Adapter Configuration Utility icon gets created on the desktop.
