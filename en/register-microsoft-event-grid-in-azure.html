Register Microsoft Event Grid in Azure

Note

This procedure is required if you are configuring Azure for Blob storage DLP and malware scanning.

For Netskope to receive events from Azure, you must register Microsoft Event Grid for your webhook subscription. A webhook (also called a web callback or HTTP push API) is a way for an app to provide other applications with real-time information.

The following procedure describes the steps to register Microsoft Event Grid in Azure for your webhook subscription.

  1. Log in to the Azure portal (https://portal.azure.com) as a global administrator.

  2. On the left navigate pane, click All Services.

  3. Under the General category, click Subscriptions.

    Note

    The following steps must be performed on a per subscription basis.

  4. Click the desired subscription from the list.

  5. In the attributes list, navigate to SETTINGS > Resource providers.

  6. In the search bar, type Microsoft.EventGrid.

  7. For the Microsoft.EventGrid provider, ensure that the STATUS is set to Registered.

You have successfully registered Microsoft Event Grid for your webhook subscription.
