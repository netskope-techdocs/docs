Widget Library

Click the Widget Library tab during report creation to add a predefined widget to your report. Use the search bar to enter a widget name or tag for easy access. 

There are several types of tags that you can click to view only the applicable widgets. For example, click the Compliance tag to show only compliance related widgets in the list view. The available tags include: Applications, Compliance, Usage, Data Center, Categories, and Users.

Click the + sign to the right of the widget name to add it to your report.
