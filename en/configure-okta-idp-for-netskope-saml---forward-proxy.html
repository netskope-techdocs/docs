Configure Okta IdP for Netskope SAML – Forward Proxy

  1. Log in to OKTA console.

  2. Click Admin to go to the Admin console.

  3. Click Developer Console and select Classic UI.

  4. From the menu select Applications > Applications.

  5. Click Add Application.

  6. In the search bar, enter Netskope. This will list Netskope apps. Click Add on the first app for Netskope User Enrollment.

  7. Enter an Application Label, like Netskope FP SAML Auth. Enter the subdomain and click Done.

  8. Select Sign On (tab) and enter the Netskope Org ID you copied from the tenant UI. Set the Application username format as Email and click Save.

  9. On the Sign On tab, click View Setup Instructions and copy these settings to configure in the Netskope UI: Name, IDP URL, IDP entity ID, and IDP certificate.

  10. Select the Assignment tab and add the users or groups who will be using it to sign-in via Netskope Client (IdP Mode).

The Okta SAML Auth configuration is complete.
