Overview

This release provides several improvements for Netskope’s on-premises Cloud Access Security Broker (CASB), such as new features, enhancements, known issues, and fixed issues. The N2000 and N5000 appliances provide the physical footprint to process and maintain data inside an enterprise's perimeter.

Note

The N1000 and N10000 appliances are no longer available for purchase. Netskope continues to support software upgrades on existing N1000 and N10000 appliances.
