Activity name change

CASB Inline Protection

DLP data in email subject is not triggering the DLP policy

We have added DLP support for calendar events where information can be passed in the subject line. With this customers can create a DLP policy over creating and editing the calendar events.

Download file size policy

Sorting by compliance for Microsoft 365

  • Policies > Security Posture > Profiles & Rules

SSPM alerts for Microsoft 365 on the Skope IT > Alerts page now includes the correct app name and category so users can see the CCI and CCL on the alert detail panel.
