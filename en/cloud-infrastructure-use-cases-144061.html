Cloud Infrastructure Use Cases

The following use cases walk you through the key features of the Cloud Infrastructure pages.

Viewing the latest security assessment scan results

  1. In the  Netskope UI, go to Cloud Infrastructure > Compliance. The Compliance page lists the latest security assessment scan results of all the rules and IaaS resources.

  2. Click the Resources toggle switch to view the scan results of your IaaS resources. By default, the page displays the results of the security assessment rules.

  3. Click at the end of each row to view the details of the findings. Depending on whether you are viewing the scan results of rules or resources, click View Rule Compliance Findings or View Resource Compliance Findings.

Exporting Compliance scan and Inventory results as CSV or JSON

To export the scan results as CSV,

  1. For Compliance report, in the Netskope UI go to Cloud Infrastructure > Compliance and click Export. 

    For Inventory report, in the Netskope UI go to Cloud Infrastructure > Inventory and select the resource item from the Inventory dropdown. Click Export. 

  2. In the Export dialog box, select the number of rows you want to export.

  3. Enter a name for the CSV file and click Export.

To export the scan results as JSON, use Netskope's REST API endpoints. 

  • Use the security assessment endpoints to export the results of security assessment rules. 

    https://<tenant-name>.goskope.com/api/v1/introspection_instance

  • Use API Data Protection instance endpoint to export the results of IaaS resources.

    https://<tenant-name>.goskope.com/api/v1/introspection_instance

For more information on REST API endpoints see, Public Cloud API Endpoints.

Viewing failure details and addressing failed rules

  1. In the  Netskope UI, go to Cloud Infrastructure > Compliance. The total number of Rules Failed, Rules Passed, and Rules Checked are displayed on the Rules page.

  2. Click the Resources toggle switch to view the total number of Resources Failed, Resources Passed, and Resources Checked.

  3. Click  at the end of a row with a Failed status to view the details of the findings. Depending on whether you are viewing the scan results of rules or resources, click View Rule Compliance Findings or View Resource Compliance Findings.

    The page displays the total number of assets checked or rules checked along with the status of the asset or rule for each region.

To address the violations against the rule, click on the rule in the Rules page under Compliance and follow the Remediation Steps at the bottom of the page.

Viewing the latest inventory

  1. In the Netskope UI, go to Cloud Infrastructure > Inventory  to view widgets with the total number resources and alerts under each category.

    The following widgets are displayed on the dashboard.

    • Accounts

    • Compute

    • Storage

    • Databases

    • VPC

    • Security Groups

    • Roles

    • Users

  2. Click on a widget to view detailed information about the resources in that category.
