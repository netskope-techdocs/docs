Modify the Timezone on the OPLP

The default timezone of the OPLP is UTC. If your log files use a different timestamp, you can modify the timezone of the OPLP to match the timestamp in the log files.

In the configuration mode of nsshell, enter the command,

set system timezone <tz>

where <tz> is the name of a region/city.

Depending on the region, you can specify one of the following values to set the timezone of the OPLP.
